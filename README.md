## Real Time Traffic Plan (RTTP)

Deze repository vormt de werkomgeving voor het **Conceptueel Informatiemodel RTTP**. Hier worden het informatiemodel, de bijbehorende documentatie en ondersteunende bestanden beheerd, ontwikkeld en onderhouden.

De repository ondersteunt de samenwerking tussen domeinexperts, informatieanalisten, architecten en andere betrokkenen bij de ontwikkeling van een gemeenschappelijk begrip van de RTTP-informatiebehoefte. Het conceptueel informatiemodel beschrijft de relevante informatieobjecten, begrippen en relaties op een technologie-onafhankelijk niveau.
